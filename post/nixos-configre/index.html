Why NixOS§

Nix, pure by design. The advantages are not explained. This article more like a tutorial.

Configuration analysis§

  • my nixos config

enable flake configuration.nix

nix = {
  extraOptions = ''
    experimental-features = nix-command flakes
  '';
};

boot§

I use grub because its easy to use os prober and colorful theme.

boot.loader = {
  grub = {
    enable = true;
    device = "nodev";
    useOSProber = true; # os prober will auto add other system to grub
    efiSupport = true;
  };
  efi.canTouchEfiVariables = true;
};

network§

Simply use network-manager.

networking = {
  hostName = "whitefox";
  useDHCP = false;

  networkmanager = {
    enable = true;
    wifi.macAddress = "random";
    ethernet.macAddress = "random";
  };
};
# gui tool
programs.nm-applet.enable = true;
